# react-auth-sample

react のログイン認証サンプル

## プロジェクト作成

```bash
> npm create vite@latest
> npm install react-router-dom
> npm install
```

## プロジェクト実行

```bash
package.jsonファイルを編集
"dev": "vite",
↓
"start": "vite",
```

```bash
> npm start
```
